## 1. システム概要

**みんなの体育館 予約システム**は、以下2拠点の体育館・トレーニングルームを LINE から予約・決済できるサービスです。

| 拠点 | 所在地 | 主な施設 |
|------|------|---------|
| みんなの体育館 **ASP** | 秋田市 | 体育館・トレーニングルーム（貸切／相席） |
| みんなの体育館 **やばせ** | 秋田市八橋 | 体育館 |

- **決済完了後に4桁の入館用暗証番号（PIN）が発行**され、入口の電子ロックで入館できます。
- 複数日一括予約・定期予約（毎週／隔週）・クーポンコード割引に対応しています。
- 予約完了後は **Google カレンダーへの自動登録**、**領収書 PDF ダウンロード**、**レビュー投稿**が可能です。

### 1.1 アプリの起動方法（LINE から LIFF で開く）

本アプリは **LINE ミニアプリ（LIFF）** として動作します。LINE 公式アカウントから配信される **LIFF URL** をタップすると、LINE アプリ内で自動的に起動します。通常のブラウザにアプリを別途インストールする必要はありません。

**起動手順**

1. LINE で「みんなの体育館」公式アカウントを**友だち追加**します。
2. 公式アカウントから届くメッセージや、トーク画面下部のメニュー（リッチメニュー）にある**予約リンク（LIFF URL）をタップ**します。
3. LINE アプリ内で予約画面が自動的に開きます。初回はログイン認証が表示されるため、「許可する」を選択してください。

> 📱 **LIFF URL**: <https://liff.line.me/2008793930-iDPTCREA>
> 上記リンクを LINE 内でタップすれば直接アプリを起動できます。

> ℹ️ **ブラウザから直接開くことも可能**ですが、LINE 認証が必要なため**LINE アプリからの起動を推奨**します。外部ブラウザで開いた場合はログインを求められることがあります。

## 2. 利用者向けマニュアル

### 2.1 拠点の選択

アプリを開くと、最初に利用したい拠点を選ぶ画面が表示されます。**みんなの体育館は複数拠点あり、利用したい場所をここで切り替えられます。**

![拠点選択画面](checkin/1.png)

**操作手順**

1. 「みんなの体育館 **ASP**」または「みんなの体育館 **やばせ**」をタップして拠点を選択します。
2. 選択後、画面下部の「**次へ進む**」ボタンをタップして施設選択に進みます。

> ℹ️ **管理画面リンクについて**
> 現在はデモ版として画面下部に「管理画面」へのリンクを表示していますが、**公開版では非表示**になります。一般利用者は管理画面にはアクセスできません。

---

### 2.2 施設と料金の確認

拠点を選択すると、その拠点で利用可能な施設の一覧と料金表が表示されます。拠点によって**利用できる施設と料金が異なる**ため、必ずここで内容を確認してください。

![施設選択画面](checkin/2.png)

**操作手順**

1. 利用したい施設をタップ（選択）します。
   - **体育館** — バスケットボール・バレーボール等（07:00 - 21:00）
   - **トレーニングルーム（貸切）** — 貸切でトレーニング機器をご利用（07:00 - 21:00）
   - **トレーニングルーム（相席）** — 他の利用者と共有でご利用（07:00 - 21:00）
2. 画面下部の「**料金表（税込）**」で時間帯別の料金を確認できます。
3. 施設を選んだら「**次へ進む**」をタップします。

> 🔁 **拠点ごとにメニューが異なります**
> やばせ拠点では「体育館」のみ提供しています（トレーニングルームは ASP のみ）。

---

### 2.3 単日予約

施設を選ぶと「日時選択」画面に移ります。1日だけ予約する場合は「**単日予約**」タブを使用します（初期選択済み）。

![単日予約画面](checkin/3.png)

**操作手順**

1. 画面上部で「**単日予約**」タブが選択されていることを確認します。
2. 「**利用日**」欄のカレンダーアイコンをタップし、予約したい日付を選びます。
3. 「**開始時間**」プルダウンから開始時刻を選択します（施設営業時間内の1時間単位）。
4. 「**利用時間**」プルダウンから利用時間を選択します（1〜4時間）。
5. 画面下部に自動計算された「**お支払い金額**」が表示されます。
6. 「**確認画面へ**」をタップして次のステップに進みます。

---

### 2.4 複数日予約・定期予約

同じ曜日・時間帯で**複数日まとめて予約**したり、**毎週／隔週の定期予約**を取ることができます。

![複数日予約画面](checkin/4.png)

**操作手順**

1. 画面上部の「**複数日予約**」タブをタップします。
2. *(任意)* 「**定期予約（任意）**」エリアで「毎週」または「隔週」を選ぶと、間隔を指定できます。
3. 「**利用日（複数選択可）**」の日付チップ（今日・明日・4/18(土) 〜 ）を**複数タップ**して選択します。
4. 「**開始時間**」「**利用時間**」を選択します。すべての選択日で同じ時間が適用されます。
5. 画面下部に**全日分合算**した「お支払い金額」が表示されます。
6. 「**確認画面へ**」をタップして次のステップに進みます。

> 💡 **定期予約を使うと**、基準日を選ぶだけで、毎週／隔週で指定回数分の予約を自動的に登録できます。

---

### 2.5 確認・クーポン・お支払い

予約内容の最終確認、クーポンコード入力、決済方法の選択を行います。

![ご利用内容確認画面](checkin/5.png)

**画面の構成**

| エリア | 内容 |
|------|------|
| **施設・利用日・利用時間** | 選択した予約内容の確認 |
| **施設利用料・お支払い金額** | 料金の内訳表示 |
| **クーポンコード** | 持っているクーポンコードを入力 |
| **ご利用にあたって** | 暗証番号・キャンセルポリシー等の注意事項 |
| **お支払い方法** | クレジットカード決済（Stripe） |

**操作手順**

1. 予約内容に誤りがないかを確認します。
2. *(任意)* クーポンコードをお持ちの場合は「**クーポンコード**」欄に入力し、「**適用**」ボタンをタップします。割引が自動的に反映されます。
3. 注意事項を確認します（下記4点）。
   - 決済完了後、入館用の4桁暗証番号が発行されます
   - 暗証番号は当日の利用時間のみ有効です
   - 入口の電子ロックに暗証番号を入力してください
   - キャンセルは利用開始1時間前まで可能です
4. 「**カードで支払う**」ボタンをタップすると、Stripeの決済画面に移動します。画面の指示に従いクレジットカード情報を入力して決済を完了します。

### 2.6 予約完了・入館方法

決済が完了すると、以下の完了画面が表示されます。

![予約完了画面](checkin/6.png)

**画面に表示される情報**

- ✅ **入館用暗証番号**（4桁の数字） — 大きく表示されるため、覚えておくか「**コピーする**」で控えてください。
- 📅 **Google カレンダーに追加** — タップすると、Google カレンダーに予約を自動登録できます（PIN もメモ欄に自動入力）。
- 📖 **入館方法**（3ステップ）
  1. 玄関の電子ロックに暗証番号を入力して入館
  2. 体育館の電子ロックに同じ暗証番号を入力
  3. ロックが解除されたらご利用開始
- 📋 **予約サマリー**（拠点・利用日・利用時間・料金）

**画面下部のアクション**

| ボタン | 用途 |
|------|------|
| ⭐ レビューを書く | 利用後の感想・評価を投稿 |
| 📄 領収書ダウンロード | PDF 形式の領収書をダウンロード |
| 🏠 ホームに戻る | トップページに戻る |

> ⚠️ **暗証番号は利用時間外は無効**になります。忘れずにメモまたはカレンダー登録してください。

## 3. 管理者向けマニュアル

管理画面は `/admin` にアクセスすることで利用できます（管理者として登録された LINE アカウントのみアクセス可）。画面上部のタブで「**カレンダー / イベント / スクール / 売上**」を切り替えられます。

### 3.1 カレンダー（予約の確認・追加・削除）

月表示カレンダーで、予約（青）・イベント（橙）・スクール（緑）を一覧表示します。**管理者は予約を直接追加・削除できます。**

![管理画面カレンダー](checkin/7.png)

**画面の見方**

- 上部に「**◀ 年月 ▶**」で月を切り替え、「**今日**」ボタンで現在月に戻れます。
- カレンダーのマスには、その日に入っている「予約●件 / イベ●件 / スク●件」のバッジが色分けで表示されます。
- マスをタップすると**下部に詳細パネル**が開き、その日の予定一覧が表示されます。

**予約を追加する**

1. カレンダー右上の「**+ 予約を追加**」ボタン、または日付を選択した状態で下部詳細の「**この日に追加**」をタップします。
2. モーダルで以下を入力します。
   - 拠点（ASP / やばせ）
   - 施設（体育館 / トレーニング貸切 / トレーニング相席）
   - 日付・開始時刻・利用時間（1〜4h）
   - *(任意)* 利用者名 — 既存ユーザーの displayName を入れると紐付け、未入力の場合は管理者名義
3. 自動計算された料金を確認し、「**予約を登録**」をタップします。
4. PIN が自動発行され、決済済（PAID）ステータスで保存されます。

**予約を削除する**

1. 日付をタップして詳細パネルを開きます。
2. 「予約」セクションの各カード右端の🗑️**ゴミ箱アイコン**をタップします。
3. 確認ダイアログで「**OK**」を選ぶと削除されます（取り消し不可）。

> ⚠️ **削除は取り消しできません。** キャンセル扱いにしたい場合は、誤削除しないように十分ご注意ください。

### 3.2 イベント管理

単発イベント（1回限り、日付指定）を作成・管理します。

![管理画面イベント](checkin/8.png)

**操作手順**

1. タブ「**イベント**」を選択します。
2. 画面左上の「**+ イベント作成**」ボタンをタップします。
3. 表示されたフォームで以下を入力します。
   - タイトル・説明
   - 拠点・施設種別
   - 開催日・開始〜終了時刻
   - 定員・料金
   - *(任意)* 画像 URL
4. 保存すると、利用者向けのイベント一覧（公開画面）に表示されるようになります。
5. 作成済みイベントは一覧で確認でき、編集・削除・参加者一覧の表示が可能です。

> 📌 イベントがまだ1件も登録されていない場合は「**イベントがありません**」と表示されます（上の画像の状態）。

---

### 3.3 スクール管理

定期開催の教室・レッスン（毎週同じ曜日・時間で複数回開催）を作成・管理します。

![管理画面スクール](checkin/9.png)

**操作手順**

1. タブ「**スクール**」を選択します。
2. 画面左上の「**+ スクール作成**」ボタンをタップします。
3. フォームで以下を入力します。
   - タイトル・説明・インストラクター名
   - 拠点・施設種別
   - 曜日（月〜日）・開始〜終了時刻
   - 定員・1回あたり料金・総回数
   - 開始日・終了日
4. 保存すると、カレンダー上で毎週の開催日に**緑バッジ**として自動展開表示されます。
5. 利用者は一括申込（全回数分の料金を一度に支払い）ができます。

> 📌 スクールがまだ1件も登録されていない場合は「**スクールがありません**」と表示されます（上の画像の状態）。

---

### 3.4 売上管理

確定済み（PAID）予約の売上を集計・可視化します。

![管理画面売上](checkin/10.png)

**画面の見方**

- 最上部のタブ「**日別 / 月別 / 拠点別**」で集計単位を切り替えられます。
- 青いカードに「**合計売上**」と「件数」が大きく表示されます。
- その下に、集計単位ごとの一覧（日付ごと、または拠点ごと）と**棒グラフのバー**で視覚的に確認できます。

**操作手順**

1. タブ「**売上**」を選択します。
2. 集計単位タブ（日別 / 月別 / 拠点別）をタップして切り替えます。
3. 必要に応じて期間（開始日・終了日）を指定して絞り込みます。
4. 各行をタップすると、その日／月／拠点の詳細内訳を確認できます（実装状況に応じて）。

> 💰 **集計対象**: ステータスが `PAID`（決済済）のチェックインのみ。`PENDING`（決済途中）は含まれません。
