【1. 定期利用ユーザーの決済画面を「請求書払い」に切替】
・`discountType = FREE`（S-01 等）の利用者には、決済画面で **クレジットカード／PayPay は表示されません**。
・「請求書払い（後日請求）」カードのみ表示し、ボタン文言も「予約を確定する」に変更。
・フッター金額表示も「¥xxx」ではなく「請求書払い」と明示。
・該当ユーザーは Stripe を経由せず、確定と同時に PIN が発行されます（オンライン決済の課金は一切発生しません）。

【2. 祝日対応】
・`@holiday-jp/holiday_jp` を導入し、日本の祝日を自動判定。
・以下が祝日に「土日祝」として扱われます。
  - 料金計算（`src/lib/price.ts` / `api/prices/calculate.ts` / `api/payments/create-checkout.ts`）→ WEEKEND 料金が自動適用
  - 売上 CSV の `時間帯区分` 列 → 祝日も「土日祝」へ分類
・例: 2026/05/05（こどもの日・火曜）→ 自動的に WEEKEND 料金、CSV 上は「土日祝」。
